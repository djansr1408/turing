# turing
Turing assignment


Installation instructions:
Navigate to **turing** directory
```
python3 -m venv env
```
```
source env/bin/activate
```
```
pip install -r requirements.txt
```

Create folder data inside turing repo and download from http://insideairbnb.com/get-the-data.html:
- listings.csv
- reviews.csv
- neighbourhoods.csv

Create folder data/zipped, download from http://insideairbnb.com/get-the-data.html:
- listings.csv.gz
- reviews.csv.gz
- calendar.csv.gz 

and extract them.

After that you can run:
```
jupyter notebook
```
